# RU Liquidity Sentinel

Система раннего предупреждения стресса ликвидности рублевого денежного рынка. Кейс ПСБ.

Автоматически собирает публичные данные ЦБ, Минфина, Росказны и ФНС, рассчитывает пять модульных сигналов и объединяет их в единый **Liquidity Stress Index (LSI)** — числовой индикатор от 0 до 100.

## Статус

Прототип готов. Реализованы все модули, ML-пайплайн, бэктест и дашборд с LLM-аналитиком.

- Данные: **2014-02-03 — 2026-05-08** (3077 наблюдений, 106 признаков)
- LSI Local на последнюю дату: **53.44** (ЖЕЛТЫЙ — повышенное внимание)
- LSI Global на последнюю дату: **38.39** (ЖЕЛТЫЙ — повышенное внимание)

## Liquidity Stress Index (LSI)

LSI рассчитывается двумя моделями на базе Isolation Forest + PCA:

| Индекс | Обучающее окно | Назначение |
|--------|---------------|-----------|
| **LSI Local** | скользящие 365 дней | чувствителен к текущим аномалиям |
| **LSI Global** | вся история с 2014 | исторически откалиброван |

### Пороговые профили

| Профиль | Зелёный | Жёлтый | Красный | Когда применять |
|---------|---------|--------|---------|----------------|
| `backtest_sensitive` (default) | < 30 | 30–60 | ≥ 60 | приоритет раннего предупреждения |
| `conservative` | < 40 | 40–70 | ≥ 70 | снижение ложных тревог |

Профиль выбирается на странице «Обзор системы» и влияет на весь дашборд.

### Результаты бэктеста

| Стрессовый эпизод | LSI Global (max) | Статус |
|------------------|-----------------|--------|
| Декабрь 2014 | ≥ 60 (1 красный день) | детектируется |
| Февраль–апрель 2022 | 100.0 (41 красный день) | детектируется |
| Август 2023 | повышен | детектируется |

Event Recall Red = 100% по Global при профиле `backtest_sensitive`.

## Модули системы

### M1 — Обязательные резервы и RUONIA

Анализирует поведение банков в периоды усреднения обязательных резервов. Спред фактических остатков vs норматива и динамика RUONIA. Ключевой драйвер LSI Local (~42% вклада на последнюю дату).

Источник: ЦБ РФ — `cbr.ru/hd_base/RReserves/`, `cbr.ru/hd_base/ruonia/`

### M2 — Аукционы РЕПО ЦБ

Спрос банков на РЕПО и ставки. Аномальный спрос (флаг `m2_Flag_Demand`) — признак дефицита краткосрочной ликвидности. Данные разреженные (только в дни аукционов).

Источник: ЦБ РФ

### M3 — Аукционы ОФЗ

Cover ratio и yield spread на первичных аукционах ОФЗ. Низкий спрос (`m3_Flag_Nedospros`) — сигнал нехватки свободных средств у участников рынка. Данные разреженные.

Источник: Минфин России

### M4 — Налоговый календарь

Детерминированный контекстный модуль на основе налогового календаря ФНС. Флаги налоговых дней, предналоговых периодов и конца квартала. Не отражает рублёвые объёмы — только календарное давление.

Источник: ФНС — [nalog.gov.ru/rn77/calendar/](https://www.nalog.gov.ru/rn77/calendar/)

### M5 — Ликвидность и Казначейство

Структурная позиция ликвидности банковского сектора (профицит/дефицит), бюджетные средства Минфина в банках и чистые потоки Росказны. Флаг `m5_Flag_Budget_Drain` — ключевой сигнал бюджетного изъятия.

Данные Росказны доступны с 2021 года.

Источники: ЦБ РФ, Минфин, Росказна

## LLM-аналитик

Страница «Аналитик» работает в двух режимах:

**Rule-based** (всегда доступен, без API):
- автокомментарий по текущему LSI
- флаги событий по модулям
- интерпретация статуса

**LLM** (опционально, через `.env`):
- ответы на вопросы по данным (`«Что было в феврале 2022?»`)
- retrieval по историческим периодам
- модель: Qwen3.5-122b через OpenAI-совместимый endpoint

Настройка:

```bash
cp .env.example .env
# .env:
OPENAI_API_KEY=sk-...
LLM_BASE_URL=https://llm.lambda.coredump.ru/v1
OPENAI_MODEL=qwen3.5-122b
```

Без ключа дашборд работает в rule-based режиме — ничего не ломается.
