# М5: средства Казначейства и бюджетные средства в банках

## Статус

Backend pipeline М5 подготовлен на базовом уровне: скачивает официальные данные ЦБ, парсит source-level CSV и собирает `m5_dataset.csv`

Feature dataset для ML пока не собирается. Его нужно добавлять после анализа `m5_dataset.csv` аналитиком/ML

## Источники

1. Бюджетные средства на счетах кредитных организаций, Банк России  
   `https://www.cbr.ru/vfs/statistics/banksector/borrowings/02_29_Budget_all.xlsx`

2. Ликвидность банковского сектора, Банк России  
   `https://www.cbr.ru/hd_base/bliquidity/`

Источник Росказны по размещению средств ЕКС на банковских депозитах пока не включен в pipeline: страница нестабильно открывается из скрипта и может зависать на SSL-подключении. Если позже получится надежно получать XLS/XLSX/PDF-файлы Росказны, их лучше добавить отдельным downloader/parser, не ломая текущий M5

## Что лежит в source CSV

`cbr_budget_funds.csv` содержит месячные остатки бюджетных средств на счетах кредитных организаций:

- `date`
- `currency_type`
- `budget_funds_total_mln_rub`
- `federal_budget_funds_mln_rub`
- `regional_local_budget_funds_mln_rub`
- `other_budget_funds_mln_rub`
- `extra_budgetary_funds_mln_rub`
- `source_url`
- `source_file`

`currency_type` принимает значения:

- `rub` — рублевый лист Excel
- `foreign_currency_and_precious_metals` — иностранная валюта и драгоценные металлы
- `total` — итоговый лист Excel

`cbr_liquidity.csv` содержит дневную таблицу ликвидности банковского сектора ЦБ:

- `date`
- `liquidity_deficit_surplus_bln_rub`
- `liquidity_deficit_surplus_without_correspondent_accounts_bln_rub`
- `cbr_claims_standard_instruments_bln_rub`
- `repo_fx_swap_auctions_bln_rub`
- `secured_loans_auctions_bln_rub`
- `repo_fx_swap_standing_bln_rub`
- `secured_loans_standing_bln_rub`
- `cbr_liabilities_standard_instruments_bln_rub`
- `deposit_auctions_bln_rub`
- `deposit_standing_bln_rub`
- `cobr_bln_rub`
- `nonstandard_refundable_operations_bln_rub`
- `correspondent_accounts_bln_rub`
- `required_reserves_avg_bln_rub`
- `source_url`
- `source_file`

## Как собран m5_dataset.csv

`m5_dataset.csv` собирается по дневной таблице ликвидности ЦБ

К каждой дневной дате добавляется последняя известная месячная запись `currency_type = total` из `cbr_budget_funds.csv`, дата которой не позже текущей даты

Это техническое объединение источников разной частоты:

- бюджетные средства — месячная частота
- ликвидность банковского сектора — дневная частота

## Важные решения

- В `m5_dataset.csv` не считаются стресс-сигналы, MAD-score, LSI и ML-признаки
- Денежные показатели из файла бюджетных средств ЦБ оставлены в млн руб., как в источнике
- Денежные показатели таблицы ликвидности ЦБ оставлены в млрд руб., как в источнике
- `budget_funds_total_mln_rub` — это остаток бюджетных средств на счетах кредитных организаций, а не объем аукционов Казначейства
- Итоговый feature builder М5 нужно делать отдельно после ноутбука аналитика
